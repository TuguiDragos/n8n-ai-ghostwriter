# n8n-ai-ghostwriter
  AI Ghostwriter for n8n: an autonomous content engine that researches a topic with live sources, writes and fact-checks a ~1,000-word article, generates a cover image, drafts social posts, and auto-publishes to Ghost + Facebook, Threads, X &amp; Bluesky. Powered by GPT-5.5, Notion and Cloudflare R2.
